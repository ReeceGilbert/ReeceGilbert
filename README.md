## Featured Project

### Nearoh Coding Language

Nearoh is a Python-inspired programming language built from scratch in C.

It currently supports:

- Lexing and parsing
- AST execution
- Variables
- Loops
- Functions
- Classes and objects
- Lists
- Runtime behavior
